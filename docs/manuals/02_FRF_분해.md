# 메뉴 2. FRF 분해 (전달함수) — 사용 매뉴얼

## 개요

ODE (운동방정식)를 입력하면 전달함수 G(s)를 구성하고, 부분분수 분해를 수행하여
1차/2차 서브시스템의 특성 파라미터(omega_b, omega_n, zeta)를 추출합니다.

## 입력 흐름

### Step 1: 입력 방식 선택

```
입력 방식 (a/b) [a]:
```

- **a) ODE 문자열** — 사람이 읽는 형태 그대로 입력
- **b) 계수 리스트** — 숫자만 쉼표로 구분하여 입력

### Step 2-a: ODE 문자열 입력 (방식 a 선택 시)

```
ODE 입력: x''' + 3x'' + 6x' + 8x = f(t)
```

**지원하는 표기법:**

| 표기 | 예시 | 의미 |
|------|------|------|
| 프라임(') | `x'''`, `x''`, `x'` | 3차, 2차, 1차 미분 |
| 캐럿 | `x^(3)`, `x^(2)`, `x^(1)` | 동일 |
| 캐럿 (괄호 없이) | `x^3`, `x^2` | 동일 |

**계수 규칙:**
- 계수 생략 시 1로 간주: `x'''` = `1*x'''`
- 음수 가능: `-2x'' + 3x' - x`
- 소수 가능: `1.2x'' + 4.2x' + 4x`
- 우변 `= f(t)` 부분은 무시됨 (안 써도 됨)

**예시들:**
```
x''' + 3x'' + 6x' + 8x = f(t)
x'' + 0.2x' + 4x = f(t)
x''' + 1.2x'' + 4.2x' + 4x = F(t)
2x'' + x' + 10x = u(t)
```

### Step 2-b: 계수 리스트 입력 (방식 b 선택 시)

```
계수 입력 (고차->저차, 쉼표 구분): 1,3,6,8
```

- **고차항부터 저차항 순서로** 입력
- `x''' + 3x'' + 6x' + 8x` → `1,3,6,8`
- `2x'' + x' + 10x` → `2,1,10`

### Step 3: 분자 계수 (선택사항)

```
분자 계수 (Enter for [1]):
```

- 대부분 Enter (기본값 [1])로 넘김
- 분자가 `s + 2` 같은 경우: `1,2` 입력

### Step 4: 강제 입력 f(t) (선택사항)

```
강제 입력 f(t) (Enter to skip):
  예: cos, 3cos2, 5sin(3), 2cos(1)+3sin(4)
f(t) = 3cos(2)
```

- Enter로 건너뛰면 강제응답 계산을 하지 않음
- 조화입력: cos/sin → 정상상태 응답 (크기/위상)
- 특수입력: impulse/step/ramp/exp → 역라플라스 변환으로 x(t) 전체 응답

**지원 형식 (10가지 예시):**

| # | 입력 | 의미 | 응답 계산 방법 |
|---|------|------|---------------|
| 1 | `impulse` | f(t) = δ(t) (단위 충격) | x(t) = L⁻¹{G(s)} |
| 2 | `3impulse` | f(t) = 3δ(t) | x(t) = 3·L⁻¹{G(s)} |
| 3 | `step` | f(t) = u(t) (단위 계단) | x(t) = L⁻¹{G(s)/s} |
| 4 | `5step` | f(t) = 5·u(t) | x(t) = 5·L⁻¹{G(s)/s} |
| 5 | `ramp` | f(t) = t·u(t) (램프) | x(t) = L⁻¹{G(s)/s²} |
| 6 | `exp(-2t)` | f(t) = e^(-2t) | x(t) = L⁻¹{G(s)/(s+2)} |
| 7 | `cos` | f(t) = cos(t) | x_ss = \|G(j)\|cos(t + ∠G) |
| 8 | `3cos(2)` | f(t) = 3cos(2t) | x_ss = 3\|G(j2)\|cos(2t + ∠G) |
| 9 | `5sin(3)` | f(t) = 5sin(3t) | x_ss = 5\|G(j3)\|sin(3t + ∠G) |
| 10 | `2cos(1)+3sin(4)` | f(t) = 2cos(t)+3sin(4t) | 중첩 원리로 합산 |

**기타 지원:**
- `delta` = `impulse`
- `u(t)` = `step`
- `2t` = `2ramp`
- `3exp(-0.5t)` = 스케일된 지수감쇠
- `10` (숫자만) = `10step` (상수 입력 = 스케일된 계단함수)

## 출력 내용

솔버가 자동으로 계산하는 항목:

| 항목 | 설명 |
|------|------|
| 전달함수 G(s) | 분모/분자 다항식 표현 |
| 특성근 (poles) | 분모 다항식의 근 |
| 부분분수 분해 | 1차 + 2차 서브시스템으로 분리 |
| 1차 서브시스템 | break frequency omega_b, 시정수 tau |
| 2차 서브시스템 | 고유진동수 omega_n, 감쇠비 zeta |
| **G₁(jω), G₂(jω)** | **각 서브시스템의 jω 도메인 함수 (실수부/허수부 분리)** |
| **특성값 테이블** | **G, G₁, G₂ 각각의 \|G(0)\|, \|G(jω_b)\|, \|G(jω_n)\|, \|G(j∞)\|** |
| **강제응답 x(t)** | **f(t) 입력 시 정상상태 응답 x_ss(t) 계산 (선택)** |
| 안정성 판별 | 모든 극점이 좌반면에 있는지 확인 |

### jω 도메인 출력 상세

각 서브시스템에 대해 s = jω를 대입한 주파수 응답 함수를 출력합니다:

**1차 서브시스템 예시** (G₁(s) = 1/(6(s+2))):
```
G₁(jω) = [1] / [(12) + j*(6ω)]
Re[G₁(jω)] = 1 / (3(ω² + 4))
Im[G₁(jω)] = -ω / (6ω² + 24)
```

**2차 서브시스템 예시** (G₂(s) = -(s-1)/(6(s²+s+4))):
```
G₂(jω) = [(1) + j*(-ω)] / [(24 - 6ω²) + j*(6ω)]
Re[G₂(jω)] = (2 - ω²) / (3(ω⁴ - 7ω² + 16))
Im[G₂(jω)] = ω(ω² - 5) / (6(ω⁴ - 7ω² + 16))
```

이를 통해 특정 주파수에서의 크기/위상을 직접 계산하거나,
Bode plot을 손으로 그릴 때 활용할 수 있습니다.

### 특성값 테이블 출력 상세

G(전체), G₁(1차), G₂(2차) 각각에 대해 세 특성 주파수에서의 값을 자동 계산합니다:

```
G:
  |G(0)| = 0.125                        ← DC (ω=0)
  |G(jω_b)| = 0.176777  (∠ -135.00°)   ← break frequency
  |G(jω_n)| = 0.176777  (∠ -135.00°)   ← natural frequency
  |G(j∞)| = 0                           ← 고주파 극한

G1:
  |G1(0)| = 0.0833333
  |G1(jω_b)| = 0.0589256  (∠ -45.00°)
  ...

G2:
  |G2(0)| = 0.0416667
  |G2(jω_b)| = 0.186339  (∠ -153.43°)
  ...
```

### 강제응답 출력 상세

f(t) 입력 시 정상상태 응답 x_ss(t)를 계산합니다.

**원리:** x_ss(t) = |G(jω₀)| * F₀ * cos(ω₀t + ∠G(jω₀))

```
Forced response: f(t) = 2 cos(2 t)
  G(j2) = -0.125 -0.125j
  |G(j2)| = 0.176777
  ∠G(j2) = -135.00°

  G1(j2) : |G1| = 0.0589256, ∠G1 = -45.00°
  G2(j2) : |G2| = 0.186339, ∠G2 = -153.43°

  x_ss(t) = 0.353553 cos(2t -135.00°)
```

다중 조화입력(중첩)의 경우 각 성분별로 계산 후 합산합니다.

## 사용 예제

### 예제 1: 2025 기출 — 3차 시스템

```
입력 방식 (a/b) [a]: a
ODE 입력: x''' + 3x'' + 6x' + 8x = f(t)
분자 계수 (Enter for [1]): (Enter)
```

**결과:**
- 극점: s = -2 (실수근 1개), s = -0.5 +/- j*sqrt(3.75) (복소근 1쌍)
- G1(s) = 1/(6(s+2)): 1차, omega_b = 2 rad/s
  - G₁(jω) = [1] / [(12) + j*(6ω)]
- G2(s) = -(s-1)/(6(s²+s+4)): 2차, omega_n = 2 rad/s, zeta = 0.25
  - G₂(jω) = [(1) + j*(-ω)] / [(24 - 6ω²) + j*(6ω)]
- DC gain = 1/8 = 0.125

### 예제 2: 2024 기출 — 3차 시스템

```
입력 방식 (a/b) [a]: b
계수 입력 (고차->저차, 쉼표 구분): 1,1.2,4.2,4
분자 계수 (Enter for [1]): (Enter)
```

**결과:**
- 극점: s = -1, s = -0.1 +/- j*1.9974...
- G1(s): 1차, omega_b = 1 rad/s
- G2(s): 2차, omega_n = 2 rad/s, zeta = 0.05

### 예제 3: 2차 시스템 (SDOF)


**결과:**
- 2차 서브시스템 1개: omega_n = 2 rad/s, zeta = 0.1

## 주의사항

1. **ODE 좌변만 파싱됨** — 우변 `f(t)`는 무시됩니다
2. **최고차항 계수가 0이면 안 됨** — `0*x''' + 3x'' + ...` 같은 입력 X
3. **n차 ODE는 반드시 1개 이상의 실수근** (n이 홀수일 때) → 1차+2차 분해가 보장됨
4. 분자 계수는 보통 `[1]`이면 충분 (시험 문제 기준)
