# 메뉴 4. 2-DOF Modal Analysis — 사용 매뉴얼

## 개요

M(질량행렬)과 K(강성행렬)를 입력하면 고유값 문제 K u = lambda M u 를 풀어
고유진동수, 질량정규화 모드형상, 직교성 검증, 초기조건 응답을 계산합니다.

N-DOF 시스템에 대해 범용적으로 동작합니다 (2-DOF에 제한되지 않음).

## 입력 흐름

### Step 1: 질량행렬 M

```
M (질량행렬): 9,0;0,9
```

**행렬 입력 형식:**
- 행은 `;`으로 구분
- 열은 `,`으로 구분
- 공백 무관

**예시:**

| 행렬 | 입력 |
|------|------|
| `[[9,0],[0,9]]` | `9,0;0,9` |
| `[[4,0],[0,1]]` | `4,0;0,1` |
| `[[1,0,0],[0,2,0],[0,0,3]]` | `1,0,0;0,2,0;0,0,3` |

> M은 대칭 양정치(positive-definite) 행렬이어야 합니다.

### Step 2: 강성행렬 K

```
K (강성행렬): 72,-36;-36,72
```

**예시:**

| 시스템 | K 입력 |
|--------|--------|
| wall-k1-m1-k2-m2-wall | `k1+k2, -k2; -k2, k2+k3` |
| 2025 기출 | `72,-36;-36,72` |

> K는 대칭 양반정치(positive semi-definite) 행렬이어야 합니다.
> 자유-자유 시스템의 경우 K가 양반정치이며 rigid-body mode(omega=0)가 존재합니다.

### Step 3: 초기 변위 q(0) (선택)

```
초기 변위 q(0) (예: 1,0) (Enter to skip): 1,0
```

- Enter로 건너뛰면 `[0, 0, ...]`
- 쉼표로 구분: `1,0` → q1(0)=1, q2(0)=0

### Step 4: 초기 속도 qdot(0) (선택)

```
초기 속도 qdot(0) (예: 0,0) (Enter to skip): 0,0
```

- Enter로 건너뛰면 `[0, 0, ...]`

## 출력 내용 (자동 계산 순서)

| Step | 내용 |
|------|------|
| 1-2 | M, K 행렬 표시 |
| 3 | 일반화 고유값 문제 K u = lambda M u 설정 |
| 4 | **고유값 lambda_i** 계산 |
| 5 | **고유진동수 omega_i = sqrt(lambda_i)** |
| 6 | 원시 고유벡터 |
| 7 | **질량정규화 고유벡터** u_i (u_i^T M u_i = 1) |
| 8 | 모달행렬 U = [u_1, u_2, ...] |
| 9 | **직교성 검증**: U^T M U = I (항등행렬 확인) |
| 10 | **직교성 검증**: U^T K U = Lambda (대각행렬 확인) |
| 11 | **모달 초기조건**: eta(0) = U^T M q(0), eta_dot(0) = U^T M qdot(0) |
| 12 | **모달 응답**: eta_r(t) = eta_r(0)cos(omega_r t) + [eta_dot_r(0)/omega_r]sin(omega_r t) |
| 13 | **물리 응답**: q(t) = U eta(t) = sum u_r eta_r(t) |
| 14 | 최종 물리 응답 식 (각 q_i(t) 전개) |

## 사용 예제

### 예제 1: 2025 기출 — 대칭 2-DOF

```
M (질량행렬): 9,0;0,9
K (강성행렬): 72,-36;-36,72
초기 변위 q(0): 1,1
초기 속도 qdot(0): 0,0
```

**결과:**
- lambda_1 = 4, lambda_2 = 12
- omega_1 = 2 rad/s, omega_2 = 2*sqrt(3) rad/s
- u_1 = [1/sqrt(18), 1/sqrt(18)] (동위상)
- u_2 = [1/sqrt(18), -1/sqrt(18)] (역위상)
- U^T M U = I 확인

### 예제 2: 비대칭 질량

```
M (질량행렬): 4,0;0,1
K (강성행렬): 12,-4;-4,4
초기 변위 q(0): 1,0
초기 속도 qdot(0): (Enter)
```

### 예제 3: 3-DOF 시스템

```
M (질량행렬): 1,0,0;0,1,0;0,0,1
K (강성행렬): 2,-1,0;-1,2,-1;0,-1,1
초기 변위 q(0): 1,0,0
초기 속도 qdot(0): (Enter)
```

### 예제 4: 자유-자유 시스템 (Rigid-body mode)

```
M (질량행렬): 1,0;0,1
K (강성행렬): 1,-1;-1,1
초기 변위 q(0): 1,-1
초기 속도 qdot(0): (Enter)
```

**결과:**
- lambda_1 = 0 (rigid-body mode), lambda_2 = 2
- omega_1 = 0 → eta_1(t) = eta_1(0) + eta_dot_1(0)*t (등속 병진)

## M, K 행렬 조립 가이드

시험에서 M, K를 직접 구해야 하는 경우가 많습니다. 참고용 공식:

### 직렬 체인 시스템: wall — k1 — [m1] — k2 — [m2] — k3 — wall

```
M = [[m1, 0],
     [0,  m2]]

K = [[k1+k2, -k2  ],
     [-k2,   k2+k3]]
```

### 자유-자유: [m1] — k — [m2]

```
M = [[m1, 0],
     [0,  m2]]

K = [[k,  -k],
     [-k,  k]]
```

## 주의사항

1. **M은 양정치** — 대각 원소가 모두 양수인 대칭 행렬
2. **K는 양반정치** — 음의 고유값이 나오면 WARNING 표시
3. **행렬 크기 일치** — M과 K의 크기, 초기조건 벡터의 길이가 모두 같아야 합니다
4. **2-DOF라는 이름이지만** 실제로는 N-DOF 지원 — 3x3, 4x4 등 아무 크기나 가능
