### 3.1 FRF 분해 (Frequency Response Function Decomposition)

#### 2025 Midterm P2
**문제:** $x''' + 3x'' + 6x' + 8x = f(t)$의 전달함수를 1차/2차 서브시스템으로 분해

**사용 기능:** `FRFSolver.solve()`
```python
from me551_solver.core.frf import FRFSolver
frf = FRFSolver()
result = frf.solve({'coefficients': [1, 3, 6, 8], 'numerator': [1]})
```

**솔버 수행 과정:**
1. 계수 리스트 → SymPy 다항식 변환 (`_coeffs_to_poly`)
2. 특성방정식의 근 계산: $s = -2, \quad s = -\frac{1}{2} \pm \frac{\sqrt{15}}{2}j$
3. 부분분수 분해 (`sympy.apart`):
   $$G(s) = \frac{1}{6(s+2)} + \frac{-(s-1)}{6(s^2+s+4)}$$
4. 서브시스템 특성 파라미터 추출 (`_classify_factor`):
   - **1차 시스템 $G_1$:** Break frequency $\omega_b = 2$ rad/s, 시정수 $\tau = 0.5$ s
   - **2차 시스템 $G_2$:** $\omega_n = 2$ rad/s, $\zeta = 0.25$
5. DC gain 검증: $G(0) = 1/8 = 0.125$
6. 고주파 기울기: $-60$ dB/decade (3차 시스템)

**풀이 결과:**
| 항목 | 값 |
|------|------|
| 전달함수 | $G(s) = \frac{1}{s^3 + 3s^2 + 6s + 8}$ |
| 인수분해 | $(s+2)(s^2+s+4)$ |
| $G_1(s)$ | $\frac{1}{6(s+2)}$, $\omega_b = 2$ |
| $G_2(s)$ | $\frac{-(s-1)}{6(s^2+s+4)}$, $\omega_n = 2$, $\zeta = 0.25$ |
| DC gain | 0.125 |

### 3.2 회전 후프 (Bead on Rotating Hoop) — 2024 Midterm P3

**사용 기능:** `RotatingHoopSolver.solve()`
```python
from me551_solver.templates.rotating_hoop import RotatingHoopSolver
solver = RotatingHoopSolver()
result = solver.solve({
    'R': 1.0, 'm': 1.0, 'Omega': 5.0, 'g': 9.81,
    'equilibrium_to_analyze': 'all'
})
```

**솔버 수행 과정:**
1. **에너지 정의** (SymPy 심볼릭):
   - $T = \frac{1}{2}m(R^2\dot{\theta}^2 + R^2\Omega^2\sin^2\theta)$
   - $V = -mgR\cos\theta$

2. **라그랑주 방정식 유도:**
   $$\ddot{\theta} + \frac{g}{R}\sin\theta - \Omega^2\sin\theta\cos\theta = 0$$

3. **평형점 탐색:**
   - Case A: $\sin\theta_{eq} = 0 \Rightarrow \theta_{eq} = 0$ (하단)
   - Case B: $\cos\theta_{eq} = \frac{g}{R\Omega^2} = 0.3924$ → $\theta_{eq} = 66.9°$ (비자명 평형)

4. **선형화 (각 평형점 주변):**
   - $\theta = 0$: $\ddot{\epsilon} + (g/R - \Omega^2)\epsilon = 0$ → $\kappa_0 = -15.19 < 0$ → **불안정**
   - 비자명 평형: $\ddot{\epsilon} + \Omega^2\sin^2\theta_{eq}\cdot\epsilon = 0$ → **안정**

5. **고유진동수:** $\omega_n = \Omega|\sin\theta_{eq}| = 4.60$ rad/s

**풀이 결과:**
| 항목 | 값 |
|------|------|
| EOM | $\ddot{\theta} + (g/R)\sin\theta - \Omega^2\sin\theta\cos\theta = 0$ |
| $\theta_{eq}$ (하단) | $\theta = 0$ → **UNSTABLE** ($\Omega > \sqrt{g/R}$) |
| $\theta_{eq}$ (비자명) | $\cos\theta_{eq} = g/(R\Omega^2) = 0.3924$ → **STABLE** |
| $\omega_n$ | $\Omega\sin\theta_{eq} = 4.60$ rad/s |

### 3.3 회전 삼각형 (Bead on Rotating Triangle) — 2025 Midterm P3

**사용 기능:** `RotatingTriangleSolver.solve()`
```python
from me551_solver.templates.rotating_triangle import RotatingTriangleSolver
solver = RotatingTriangleSolver()
result = solver.solve({
    'm': 1.0, 'Omega': 5.0, 'g': 9.81,
    'orientation': 'midterm_2025'
})
```

**솔버 수행 과정:**
1. **시스템 기하학:** 정삼각형이 연직축 주위로 $\Omega$으로 회전, 비드가 한 변을 따라 슬라이딩
2. **유효 라그랑지안:**
   - $T_{eff} = m\dot{r}^2$ (구속조건으로 인한 계수 2)
   - $V_{eff} = mgr$ (중력 성분)
3. **EOM:** $2\ddot{r} - \Omega^2 r + g = 0$
4. **평형점:** $r_{eq} = g/\Omega^2 = 0.3924$ m
5. **선형화:** $\ddot{\rho} - (\Omega^2/2)\rho = 0$ → $\kappa = -\Omega^2/2 < 0$
6. **안정성:** $\lambda = \pm\Omega/\sqrt{2} = \pm 3.536$ → 양의 실수 고유값 존재 → **불안정 (안장점)**

**풀이 결과:**
| 항목 | 값 |
|------|------|
| EOM | $2\ddot{r} - \Omega^2 r + g = 0$ |
| $r_{eq}$ | $g/\Omega^2 = 0.3924$ m |
| 선형화 EOM | $\ddot{\rho} - (\Omega^2/2)\rho = 0$ |
| 고유값 | $\lambda = \pm\Omega/\sqrt{2} = \pm 3.536$ |
| 안정성 | **UNSTABLE** (안장점 평형) |

### 3.4 2-DOF 비감쇠 모달 해석 — 2024 Midterm P4

**사용 기능:** `ModalSolver.solve()`
```python
from me551_solver.core.modal import ModalSolver
solver = ModalSolver()
result = solver.solve({
    'M': [[4, 0], [0, 4]],
    'K': [[32, -16], [-16, 32]],
    'initial_q': [1, 1], 'initial_qdot': [0, 0]
})
```

**솔버 수행 과정:**
1. **행렬 조립:** $M = 4I_2$, $K = \begin{bmatrix} 32 & -16 \\ -16 & 32 \end{bmatrix}$ (from $k_1=k_2=k_3=16$)
2. **일반화 고유치 문제:** `scipy.linalg.eigh(K, M)` 사용
   - $\lambda_1 = 4$, $\lambda_2 = 12$
3. **고유진동수:** $\omega_1 = 2$ rad/s, $\omega_2 = 2\sqrt{3} \approx 3.464$ rad/s
4. **질량 정규화 모드형상:** $u_1 = \frac{1}{2\sqrt{2}}\begin{bmatrix}1\\1\end{bmatrix}$, $u_2 = \frac{1}{2\sqrt{2}}\begin{bmatrix}1\\-1\end{bmatrix}$
5. **직교성 검증:**
   - $U^T M U = I$ → PASS
   - $U^T K U = \text{diag}(4, 12)$ → PASS
6. **모달 초기조건:** $\eta_1(0) = 2\sqrt{2}$, $\eta_2(0) = 0$
7. **모달 응답:** $\eta_1(t) = 2\sqrt{2}\cos(2t)$, $\eta_2(t) = 0$ (2번 모드 여기 안 됨)
8. **물리 응답:** $q_1(t) = q_2(t) = \cos(2t)$ (동위상 운동만 존재)

**풀이 결과:**
| 항목 | 값 |
|------|------|
| $\lambda_1, \lambda_2$ | 4, 12 |
| $\omega_1, \omega_2$ | 2, $2\sqrt{3}$ rad/s |
| Mode 1 | $[1, 1]^T$ (동위상) |
| Mode 2 | $[1, -1]^T$ (역위상) |
| $q_1(t) = q_2(t)$ | $\cos(2t)$ |

### 3.5 2-DOF 비례감쇠 모달 해석 — 2025 Midterm P4

**사용 기능:** `DampingSolver.solve()`
```python
from me551_solver.core.damping import DampingSolver
solver = DampingSolver()
result = solver.solve({
    'M': [[9,0],[0,9]], 'K': [[72,-36],[-36,72]],
    'C': [[9,-3.6],[-3.6,9]],  # C = 0.2M + 0.1K
    'alpha': 0.2, 'beta': 0.1,
    'damping_type': 'proportional',
    'initial_q': [1,1], 'initial_qdot': [0,0]
})
```

**솔버 수행 과정:**
1. **비감쇠 모달 해석** → `ModalSolver` 내부 호출
   - $\omega_1 = 2$, $\omega_2 = 2\sqrt{3}$ rad/s
2. **비례감쇠 확인:** $C = 0.2M + 0.1K$ → `_check_proportional_damping()` → PASS
3. **모달 감쇠비 계산:** $2\zeta_r\omega_r = \alpha + \beta\omega_r^2$
   - $\zeta_1 = 0.6 / (2 \times 2) = 0.15$
   - $\zeta_2 = 1.4 / (2 \times 3.464) = 0.2021$
4. **감쇠 고유진동수:**
   - $\omega_{d,1} = 2\sqrt{1 - 0.15^2} = 1.977$ rad/s
   - $\omega_{d,2} = 3.464\sqrt{1 - 0.2021^2} = 3.393$ rad/s
5. **감쇠 모달 응답:**
   - $\eta_1(t) = e^{-0.3t}[4.243\cos(1.977t) + 0.644\sin(1.977t)]$
   - $\eta_2(t) = 0$ (2번 모드 미여기)
6. **물리 응답:**
   - $q_1(t) = q_2(t) = e^{-0.3t}[\cos(1.977t) + 0.152\sin(1.977t)]$

**풀이 결과:**
| 항목 | 값 |
|------|------|
| $\alpha, \beta$ | 0.2, 0.1 |
| $\zeta_1, \zeta_2$ | 0.15, 0.2021 |
| $\omega_{d,1}, \omega_{d,2}$ | 1.977, 3.393 rad/s |
| $q_1(t) = q_2(t)$ | $e^{-0.3t}[\cos(1.977t) + 0.152\sin(1.977t)]$ |

## 4. 사용된 기능 및 모듈 상세

### 4.1 Problem Router (`core/router.py`)

문제 텍스트를 키워드 기반으로 자동 분류하는 기능입니다.

```python
from me551_solver.core.router import classify
r = classify('eigenvalue, modal analysis, natural frequency, mode shape')
# → route='modal', confidence=1.00
```

**라우팅 테스트 결과:**
| 입력 키워드 | 분류 결과 | 신뢰도 |
|------------|----------|--------|
| FRF, transfer function, break frequency | `frf` | 0.55 |
| Lagrange, kinetic energy, potential energy | `lagrange` | 0.90 |
| modal, eigenvalue, mode shape | `modal` | 1.00 |
| damping, Rayleigh, proportional | `damping` | 1.00 |
| gyroscopic, circulatory, flutter | `extended` | 1.00 |

### 4.2 FRFSolver (`core/frf.py`)

3차 이상의 ODE를 전달함수로 변환하고 부분분수 분해를 수행합니다.

**핵심 알고리즘:**
- `_coeffs_to_poly()`: 계수 리스트 → SymPy 다항식 (소수→유리수 자동 변환)
- `sympy.apart()`: 부분분수 분해
- `sympy.factor()`: 분모 인수분해
- `_classify_factor()`: 각 인수의 차수에 따라 $\omega_b$ (1차) 또는 $\omega_n, \zeta$ (2차) 추출

### 4.3 ModalSolver (`core/modal.py`)

N-DOF 비감쇠 시스템의 모달 해석을 수행합니다.

**핵심 알고리즘:**
- `scipy.linalg.eigh(K, M)`: 일반화 대칭 고유치 문제 (수치적으로 안정)
- `_fix_sign_convention()`: 고유벡터 부호 일관성 보장
- `_mass_normalize()`: $u_i^T M u_i = 1$ 질량 정규화
- 직교성 검증: $U^T M U = I$, $U^T K U = \Lambda$
- 모달 초기조건: $\eta(0) = U^T M q(0)$, $\dot{\eta}(0) = U^T M \dot{q}(0)$
- 물리 응답 재구성: $q(t) = U\eta(t)$

### 4.4 DampingSolver (`core/damping.py`)

비례/비비례 감쇠 시스템을 해석합니다.

**핵심 알고리즘:**
- `_check_proportional_damping()`: 최소자승법으로 $\alpha, \beta$ 추정 + 가환성 검증 ($M^{-1}C M^{-1}K = M^{-1}K M^{-1}C$)
- 모달 감쇠비: $2\zeta_r\omega_r = \alpha + \beta\omega_r^2$
- 감쇠 고유진동수: $\omega_{d,r} = \omega_r\sqrt{1-\zeta_r^2}$
- 비비례 감쇠: 상태공간 고유치 해석 → 복소 고유값에서 $\omega_n, \zeta$ 추출

### 4.5 RotatingHoopSolver / RotatingTriangleSolver (`templates/`)

특정 기출 유형에 특화된 템플릿 솔버입니다.

**수행 과정:**
1. 에너지 정의 (SymPy 심볼릭)
2. 라그랑주 방정식 유도
3. 평형점 탐색 (비선형 방정식 풀이)
4. 평형점 주변 선형화 (테일러 전개)
5. 선형화 계수의 부호로 안정성 판별

### 4.6 StabilitySolver (`core/stability.py`)

선형화된 시스템의 안정성을 고유값 기반으로 판별합니다.

**판별 기준:**
- $\text{Re}(\lambda) < 0$ (모두) → 점근 안정
- $\text{Re}(\lambda) \leq 0$ (단순 허수 고유값) → 한계 안정
- $\text{Re}(\lambda) > 0$ (하나라도) → 불안정

### 4.7 ReportEngine (`core/report.py`)

풀이 결과를 다양한 형식으로 렌더링합니다.

**출력 형식:**
- `render_markdown()`: GitHub-flavored Markdown + LaTeX 수식
- `render_exam_style()`: Unicode 박스 문자 기반 시험지 스타일
- `save_markdown()`: 파일로 저장

## 6. 주요 발견사항

### 6.1 기출 출제 패턴
3가지 핵심 유형이 매년 반복 출제됨:
1. **FRF 분해** (P2): 3차 ODE → 1차+2차 서브시스템 분해, $\omega_b$, $\omega_n$, $\zeta$ 추출
2. **라그랑주 역학** (P3): 에너지 정의 → EOM → 평형점 → 선형화 → 안정성
3. **모달 해석** (P4): $Ku = \lambda Mu$ → 모드형상 → 직교성 → 응답

### 6.2 솔버의 강점
- **단계별 풀이 출력:** 매 스텝마다 수식과 수치를 함께 제시 → 시험 답안 작성에 직접 활용 가능
- **자동 검증:** 직교성 검증, DC gain 일치, 안정성 교차검증 등 sanity check 내장
- **심볼릭 + 수치 하이브리드:** SymPy로 정확한 해석해 도출 + NumPy/SciPy로 수치 검증

### 6.3 회전 후프 vs 회전 삼각형
같은 라그랑주 역학 프레임워크를 사용하지만 결과가 정반대:
- **회전 후프:** 비자명 평형점 → **안정** (복원력 계수 양수)
- **회전 삼각형:** 평형점 → **불안정** (복원력 계수 음수, 안장점)
