## 5. HW3 Q3: Convolution Integral — 메뉴 11 (Duhamel 적분, 신규)

**입력:** system_order=2, m/c/k=심볼릭, force=exp_truncated, F0/a/t1=심볼릭

**솔버 출력:**

Impulse response:
```
h(t) = (1/(m·ωd)) · exp(-ζωₙt) · sin(ωd·t)
```

표준 적분 공식 (SymPy integrate 미사용):
```
∫₀ᴸ exp(σ̃τ)·sin(ωd(t-τ)) dτ
= [exp(σ̃L)·(σ̃·sin(ωd(t-L)) + ωd·cos(ωd(t-L)))
   - σ̃·sin(ωd·t) - ωd·cos(ωd·t)] / (σ̃² + ωd²)

여기서 σ̃ = ζωₙ - a
```

**Case i) 0 < t < t₁:**
```
x(t) = F₀/(m·ωd·D) · [ωd·exp(-at)
       + exp(-ζωₙt)·(-(ζωₙ-a)·sin(ωd·t) - ωd·cos(ωd·t))]

D = (ζωₙ-a)² + ωd²
```

**Case ii) t > t₁:**
```
x(t) = F₀/(m·ωd·D) · exp(-ζωₙt) · [
       -(ζωₙ-a)·sin(ωd·t) - ωd·cos(ωd·t)
       + exp((ζωₙ-a)t₁)·((ζωₙ-a)·sin(ωd(t-t₁)) + ωd·cos(ωd(t-t₁)))]
```

**정답과 비교:**
| 항목 | 솔버 | 교과서 정답 | 일치 |
|------|------|------------|------|
| h(t) | (1/(mωd))e^{-ζωₙt}sin(ωd·t) | 동일 | OK |
| 분모 D | (ζωₙ-a)²+ωd² | (ζωₙ-a)²+ωd² | OK |
| Case i 구조 | ωd·e^{-at} + e^{-ζωₙt}(sin,cos 항) | 동일 | OK |
| Case ii 구조 | e^{-ζωₙt}(적분 상한차이) | 동일 | OK |
| x(0) | 0 ✓ | 0 (zero IC) | OK |
