### P2. FRF Decomposition (10pts)

**문제:** $x''' + 3x'' + 6x' + 8x = f(t)$

```python
frf = FRFSolver()
result = frf.solve({'coefficients': [1, 3, 6, 8], 'numerator': [1]})
```

**풀이 과정:**

1. 전달함수 구성: $G(s) = \frac{1}{s^3 + 3s^2 + 6s + 8}$
2. 특성방정식의 근: $s = -2, \quad s = -\frac{1}{2} \pm \frac{\sqrt{15}}{2}j$
3. 분모 인수분해: $(s+2)(s^2+s+4)$
4. 부분분수 분해:
$$G(s) = \frac{1}{6(s+2)} + \frac{-(s-1)}{6(s^2+s+4)}$$

**결과:**

| 서브시스템 | 전달함수 | 특성값 |
|-----------|---------|--------|
| $G_1$ (1차) | $\frac{1}{6(s+2)}$ | $\omega_b = 2$ rad/s, $\tau = 0.5$ s |
| $G_2$ (2차) | $\frac{-(s-1)}{6(s^2+s+4)}$ | $\omega_n = 2$ rad/s, $\zeta = 0.25$ |

DC gain: $G(0) = 1/8 = 0.125$

### P3. Rotating Triangle (25pts)

**문제:** 정삼각형 프레임이 $\Omega$으로 회전, 비드가 한 변을 따라 슬라이딩

```python
solver = RotatingTriangleSolver()
result = solver.solve({'m': 1.0, 'Omega': 5.0, 'g': 9.81, 'orientation': 'midterm_2025'})
```

**풀이 과정:**

| 단계 | 내용 | 결과 |
|------|------|------|
| (a) EOM | 라그랑주 방정식 유도 | $2\ddot{r} - \Omega^2 r + g = 0$ |
| (b) 평형점 | $\ddot{r}=0$ 으로 설정 | $r_{eq} = g/\Omega^2 = 0.3924$ m |
| (c) 선형화 | $r = r_{eq} + \rho$ 대입 | $\ddot{\rho} - (\Omega^2/2)\rho = 0$ |
| (d) 안정성 | 선형화 계수 부호 판별 | $\kappa = -\Omega^2/2 = -12.5 < 0$ → **UNSTABLE** |

고유값: $\lambda = \pm\Omega/\sqrt{2} = \pm 3.536$ (양의 실수 고유값 → 안장점)

### P4. 2-DOF Modal Analysis + Proportional Damping (35pts)

**문제:** $m_1=m_2=9$ kg, $k_1=k_2=k_3=36$ N/m, $C = 0.2M + 0.1K$

```python
modal = ModalSolver()
result = modal.solve({
    'M': [[9,0],[0,9]], 'K': [[72,-36],[-36,72]],
    'initial_q': [1,1], 'initial_qdot': [0,0]
})

damp = DampingSolver()
result = damp.solve({
    'M': [[9,0],[0,9]], 'K': [[72,-36],[-36,72]],
    'C': [[9,-3.6],[-3.6,9]],
    'alpha': 0.2, 'beta': 0.1,
    'damping_type': 'proportional',
    'initial_q': [1,1], 'initial_qdot': [0,0]
})
```

**풀이 과정:**

| 단계 | 수행 내용 | 결과 |
|------|----------|------|
| (1) 행렬 조립 | $M=9I$, $K=\begin{bmatrix}72&-36\\-36&72\end{bmatrix}$ | — |
| (2) 고유치 문제 | $Ku = \lambda Mu$ (`scipy.linalg.eigh`) | $\lambda_1=4$, $\lambda_2=12$ |
| (3) 고유진동수 | $\omega_r = \sqrt{\lambda_r}$ | $\omega_1=2$, $\omega_2=2\sqrt{3}$ |
| (4) 모드형상 | 질량 정규화 | $u_1 \propto [1,1]^T$, $u_2 \propto [1,-1]^T$ |
| (5) 직교성 검증 | $U^TMU=I$, $U^TKU=\Lambda$ | PASS |
| (6) 비례감쇠 | $2\zeta_r\omega_r = \alpha + \beta\omega_r^2$ | $\zeta_1=0.15$, $\zeta_2=0.2021$ |
| (7) 감쇠 진동수 | $\omega_{d,r} = \omega_r\sqrt{1-\zeta_r^2}$ | $\omega_{d,1}=1.977$, $\omega_{d,2}=3.393$ |
| (8) 모달 응답 | 초기조건 변환 | $\eta_2(0)=0$ → 2번 모드 미여기 |
| (9) 물리 응답 | $q(t) = U\eta(t)$ | $q_1=q_2=e^{-0.3t}[\cos(1.977t)+0.152\sin(1.977t)]$ |
